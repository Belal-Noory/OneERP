# ONEERP – Print & Press Management Module

## Module Name
Print & Press Management

---

# Overview

The Print & Press Management module is a complete operational ERP solution for printing and press businesses integrated into the ONEERP ecosystem.

The system is designed for:
- Digital printing shops
- Offset printing presses
- Flex/banner printing businesses
- Packaging printing
- Photocopy and document centers
- Advertising print shops
- Large and small press businesses

The module should support:
- Single-user businesses
- Multi-user teams
- Multi-branch scalability
- Multi-tenant architecture (already implemented in ONEERP)

The module must follow the same architecture, coding structure, UI/UX, responsive design, navigation system, tenant management, and activation workflow used in existing ONEERP modules such as:
- Pharmacy Module
- Shop Module
- Sarafi System

# Multi-Language & Localization Requirements

## IMPORTANT

The module must fully support the ONEERP multilingual architecture.

The system currently supports:
- English
- Dari
- Pashto

## Development Requirement

No text, label, message, button, validation message, notification, modal, table header, menu item, form field, tooltip, alert, or UI component should miss translation support.

Every visible text in the module must use the centralized translation/localization system.

# Technical Requirements

## The module must:
- Follow the same i18n structure used in existing ONEERP modules
- Use centralized translation keys
- Avoid hardcoded text completely
- Support dynamic language switching
- Support RTL layouts for Dari and Pashto
- Maintain responsive layout consistency in all languages

---

# RTL Support

Since Dari and Pashto are RTL languages, the module must properly support:
- RTL layouts
- RTL tables
- RTL forms
- RTL navigation
- RTL modals
- RTL reports
- RTL invoice/quotation templates

The UI should automatically switch direction based on selected language.

# Development Warning

Missing translations are NOT acceptable.

Before deployment:
- all UI text must be reviewed
- all translation keys must be verified

The Print & Press Management module must maintain the same multilingual quality standards as the rest of ONEERP.
